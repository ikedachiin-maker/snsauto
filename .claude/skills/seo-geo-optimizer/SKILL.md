# SEO/GEO Optimizer

## Overview

従来のSEO（Google/Bing検索最適化）に加え、GEO（ChatGPT/Claude/Perplexity/Google AI OverviewsなどのAI生成回答に引用されるための最適化）を統合した次世代SEOスキル。

## When to Use

- 「SEO分析」「キーワードリサーチ」「SERP分析」と指示された場合
- 記事のSEO最適化が必要な場合
- GEO最適化（AI検索対応）が必要な場合
- 競合ドメイン分析が必要な場合
- Schema/構造化データの生成が必要な場合

## Required MCP Servers

- `dataforseo` - SEO/SERP/キーワードデータ取得
- `gsc` - Google Search Console自社データ

## 16 Functions

### SEO分析（8機能）

| # | 機能 | 説明 |
|---|------|------|
| 1 | キーワードリサーチ | 検索ボリューム、競合度、CPC推定値の取得 |
| 2 | SERP分析 | 上位10件の分析（タイトル、URL、スニペット、特徴） |
| 3 | 競合ドメイン分析 | 競合サイトのキーワード・トラフィック分析 |
| 4 | バックリンク分析 | 被リンクの質と量の評価 |
| 5 | コンテンツギャップ分析 | 競合が扱うが自社が扱わないキーワードの特定 |
| 6 | オンページSEO監査 | タイトル/ディスクリプション/H1-H6/画像alt等のチェック |
| 7 | テクニカルSEO監査 | Core Web Vitals、モバイル対応、サイトマップ確認 |
| 8 | ランキング追跡 | 指定キーワードの検索順位モニタリング |

### GEO最適化（4機能）

| # | 機能 | 説明 |
|---|------|------|
| 9 | AI引用最適化 | ChatGPT/Claude/Perplexityに引用されやすい文体・構造の提案 |
| 10 | 構造化スニペット最適化 | FAQ/HowTo/定義リスト等のGEO対応構造提案 |
| 11 | E-E-A-T強化 | 経験・専門性・権威性・信頼性の要素追加提案 |
| 12 | 引用ソース最適化 | AI検索が参照しやすいデータ・統計・出典の配置 |

### コンテンツSEO（4機能）

| # | 機能 | 説明 |
|---|------|------|
| 13 | SEOコンテンツ作成ガイド | キーワード密度、見出し構造、内部リンク提案 |
| 14 | メタタグ生成 | タイトル（60字以内）、ディスクリプション（160字以内）自動生成 |
| 15 | Schema生成 | Article/FAQ/HowTo/BreadcrumbのJSON-LD自動生成 |
| 16 | プログラマティックSEO | テンプレートベースの大量ページ生成戦略 |

## SEO Score Checklist

記事生成後に自動チェックする項目：

```
[ ] タイトルにターゲットキーワードが含まれる
[ ] タイトルが60文字以内
[ ] メタディスクリプションが160文字以内
[ ] H2にキーワードが1回以上含まれる
[ ] 画像にalt属性がある
[ ] 内部リンクが2本以上
[ ] 外部リンク（出典）が1本以上
[ ] 記事が2,000文字以上
[ ] Schema（JSON-LD）が埋め込まれている
[ ] モバイルフレンドリーな構造
```

## Usage Examples

```
「"AIエージェント 作り方"のキーワードリサーチをして」
「この記事のSEOスコアをチェックして」
「競合サイト example.com のSEO分析をして」
「この記事をGEO最適化して（AI検索に引用されやすくして）」
「FAQ Schema（JSON-LD）を生成して」
```

## Integration

- **DataForSEO MCP**: キーワード/SERP/バックリンクデータのリアルタイム取得
- **GSC MCP**: 自社サイトの検索パフォーマンス分析
- **content-research-writer**: 記事生成時のSEO自動適用
