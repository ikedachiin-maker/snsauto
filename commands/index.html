Slide Editor Command

/slide-editor MNP専用スライドエディター

Usage
CommandDescriptionExample
/slide-editor new [path] 新しいスライドデッキを作成 /slide-editor new mnpvsl/new-deck.html
/slide-editor edit [path] 既存スライドを編集 /slide-editor edit mnpvsl/vsl-slides.html
/slide-editor add [path] 既存デッキにスライド追加 /slide-editor add mnpvsl/vsl-slides.html
/slide-editor 対話モード起動 引数なしで実行
